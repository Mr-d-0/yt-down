## How To Install
* `sudo apt update`
* `sudo apt upgrade` 
* `sudo apt install python3 git`
* `git clone https://github.com/Mr-d-0/yt-down.git`
* `cd yt-down`
* `pip3 install -r requirement.txt`
* `python3 ytdown.py`
